# 🚀 快速启动指南

## 最简单的运行方式（Windows）

### 第一步：安装依赖（只需一次）

打开 PowerShell 或 CMD，进入项目目录，运行：

```bash
pip install -r requirements.txt
```

### 第二步：启动后端服务器

**方式 A：双击运行**
- 双击 `start_backend.bat` 文件
- 会打开一个黑色窗口，显示服务器运行信息
- **保持这个窗口打开！**

**方式 B：命令行运行**
```bash
python server.py
```

### 第三步：启动前端服务器

**新开一个窗口**（后端窗口保持打开），然后：

**方式 A：双击运行**
- 双击 `start_frontend.bat` 文件

**方式 B：命令行运行**
```bash
python -m http.server 8000
```

### 第四步：打开浏览器

访问：**http://localhost:8000**

---

## 📝 运行流程图

```
1. 安装依赖
   ↓
   pip install -r requirements.txt
   ↓
2. 启动后端（终端窗口 1）
   ↓
   python server.py
   ↓
   [保持运行] ← 不要关闭！
   ↓
3. 启动前端（终端窗口 2）
   ↓
   python -m http.server 8000
   ↓
   [保持运行] ← 不要关闭！
   ↓
4. 打开浏览器
   ↓
   http://localhost:8000
   ↓
   ✅ 完成！
```

---

## ⚠️ 重要提示

1. **必须同时运行两个服务器**
   - 后端服务器（端口 5000）
   - 前端服务器（端口 8000）

2. **不要关闭服务器窗口**
   - 关闭窗口 = 停止服务器
   - 需要使用时重新启动

3. **如果端口被占用**
   - 修改 `server.py` 中的端口号
   - 同时修改 `app.js` 中的 `API_BASE_URL`
